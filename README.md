# Catnip

## About

Fast and free music streaming music with modern UI. Written in Golang and React.

## Features
- [ ] Search
- [ ] Import playlist (from local)
- [x] Import playlist (from Youtube)
- [ ] Import playlist (from Spotify)
- [x] Add to queue
- [x] Shuffle, repeat
- [x] Play, pause, next, previous
- [x] Download music
- [ ] Lyrics
- [x] Dark mode
- [x] Mini player
- [ ] Statistics
- [x] Desktop app
